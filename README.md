# Last Minute

Last-minute booking marketplace (MVP). Spring Boot 3 backend + React (Vite) SPA, Postgres.

## Quick start

```bash
# Backend
cd backend
./mvnw spring-boot:run

# Frontend (new terminal)
cd frontend
pnpm install
pnpm dev
```

Backend: http://localhost:8080 · Frontend: http://localhost:5173.
